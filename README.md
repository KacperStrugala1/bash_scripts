## Few scripts for automation and monitoring apps and services ##
### 1. server-stats.sh ###
That program give you the following stats:

-Total CPU usage <br>
-Total memory usage (Free vs Used including percentage) <br>
-Total disk usage (Free vs Used including percentage) <br>
-Top 5 processes by CPU usage <br>
-Top 5 processes by memory usage <br>

### Starting program ###
```console
bash@bash:~$ ./server-stats.sh
```
